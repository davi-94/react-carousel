# React Carousel

Simple React Carousel

## Getting Started

This is a simple react.js carousel with SCSS using Hooks

### Installing

Fork this repository

```
npm i
```

## Development

```
npm start
```

## Deployment

```
npm run build
```

## Built With

* [React](https://reactjs.org/) - The web framework used
* [SASS](https://sass-lang.com/) - Dependency Management
